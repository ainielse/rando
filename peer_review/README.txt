These application export files are very basic applications for tracking a peer code review process.

f_peer_review.sql is an APEX application export with supporting objects. 
Install this application first.
The application tracks basic information about a peer code review.
The supporting objects contains a script with some sample data.

f_peer_review_admin.sql is an APEX application export. 
Install this application after installing f_peer_review.sql.
The application has basic admin features for the Peer Code review database.
